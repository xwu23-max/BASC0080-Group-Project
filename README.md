# BASC0080-Group-Project
This code is used to calculate the BSM European call option and to price SPX-based European call options and arithmetic Asian call options using Monte Carlo simulation.
These codes have been reproduced from Tzeng (2026).
Tzeng, W.T. (2026). A Modern Approach to Teaching Options Pricing Using Monte Carlo Simulation with Python. SSRN Electronic Journal. doi:https://doi.org/10.2139/ssrn.6112906.
